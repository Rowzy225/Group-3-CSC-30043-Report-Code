# CSC-30043: Loan Status Prediction Project

## Project Overview
This repository contains the code and analysis for the **CSC-30043 Data Mining & Machine Learning** module assessment at Keele University. The project involves building and evaluating machine learning models to predict loan status ("Fully Paid" vs. "Charged Off") based on financial and demographic data.
